== Description ==

The Clean Blog is a Responsive WordPress theme, carefully styled with Bootstrap, perfect for personal or company blogs.  
A modern design with a subtle splash of color and a distraction free blog text optimized for legibility.

== Frequently Asked Questions ==

= Does this theme support any plugins? =

The Clean Blog includes support for Infinite Scroll in Jetpack.

= How to create an item link with children ? (MANDATORY !) =

1. Go To Appearance > Menus.
2. Create a **Custom Link**
3. In _URL_ type **#**
4. In _Link Text_ type the label of your choice.
5. Put children links under this custom link.
6. Save your menu.
7. That's it :)

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
